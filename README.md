# sales-analysis-excel
A complete Excel-based sales analytics project with dashboards and insights.
## Contents
- `data/` — raw and cleaned datasets
- `dashboards/` — Excel files (Executive Overview, Performance Drilldown, Operational Insights)
- `images/` — screenshots of dashboards
- `notes/` — insight text, recommendations, and methodology
## Key Findings (summary)
- Standard Class drives the highest sales volume.
- Second Class is the best upsell/upgrade option.
- First Class underperforms and needs repositioning.
- A negative Sales vs Quantity trend indicates excessive discounting.
## Tools
Microsoft Excel (Power Query, PivotTables, PivotCharts), basic regression analysis, conditional formatting, slicers.
